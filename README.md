## 📘 Sobre o Curso

Neste curso estou aprendendo JavaScript do básico ao avançado, explorando tanto o desenvolvimento front-end quanto back-end.

Durante a formação, são abordados conceitos fundamentais da linguagem, lógica de programação, manipulação do DOM, orientação a objetos, programação assíncrona e desenvolvimento de aplicações completas com Node.js e Express.

Além da teoria, o curso conta com diversos exercícios práticos e projetos reais para aplicar os conhecimentos na prática.
